**Symfony integration for [AMP Toolbox for PHP](https://github.com/ampproject/amp-toolbox-php).**

## Installation

To install the bundle with Symfony Flex, use the recipe:

``` bash
$ composer require holadev/amp-toolbox-bundle
```

## Testing

``` bash
$ composer test
```
